The basic role of making the online area calculation the system is to make a programmed online
based system which will provide a simple and exchange approach to calculate area.
This system can calculate area of each department and the total area. In this project we are going
to code a simple program to calculate college are a department wise using C++.
We used the concepts of OODP that is Classes , Objects , Override , friend function ,Virtual Function, Exceptional Handling.
